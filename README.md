# Alfred 🧠🎤

Assistente virtual por voz desenvolvido em Python, inspirado na ideia de um mordomo digital.

## 🚀 Funcionalidades (V1)

- Reconhecimento de voz
- Execução de comandos no computador
- Resposta por voz

## 🎯 Comandos disponíveis

- "abra o navegador" → abre o Vivaldi
- "abra o Apple Music" → abre o Apple Music

## 🛠️ Tecnologias

- Python

## 📌 Objetivo

Criar uma assistente pessoal capaz de executar tarefas por comando de voz, evoluindo gradualmente para um sistema mais inteligente.
